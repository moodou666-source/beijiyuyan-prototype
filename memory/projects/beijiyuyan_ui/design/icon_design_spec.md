# 北京语燕 - 图标设计规范

## 一、Logo 重新设计

### 设计理念
- **北极雨燕**：候鸟，象征迁徙、探索、自由
- **指南针**：方向、指引、探索未知
- **结合**：雨燕 + 指南针 + 地图元素

### 设计方向
1. **抽象化雨燕造型**
   - 不要写实的燕子照片
   - 用几何线条勾勒燕子飞翔姿态
   - 翅膀展开形成指南针指针

2. **融入地图元素**
   - 地球经纬线
   - 路线轨迹
   - 定位标记

3. **色彩方案**
   - 主色：深蓝 #1a365d（夜空）
   - 辅色：金色 #d69e2e（指南针）
   - 点缀：橙色 #ed8936（活力）

## 二、UI 小图标设计规范

### 设计原则
1. **统一风格**：全部使用线性图标，2px 描边
2. **独特造型**：避免使用通用图标库
3. **语义清晰**：一看就懂功能
4. **细节精致**：圆角、断点、层次感

### 首页图标

## 三、实现方式

### 方案1：SVG 手绘（推荐）
- 使用 Figma/Sketch 绘制
- 导出 SVG 代码
- 支持任意缩放

### 方案2：Icon Font
- 使用 IcoMoon 生成
- 统一字体文件
- 方便颜色控制

### 方案3：PNG 切图
- 设计稿导出
- 多倍图适配
- 简单直接
