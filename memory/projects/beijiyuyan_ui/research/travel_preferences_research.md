# 北京语燕 - 旅行偏好体系研究

## 一、旅行偏好分类体系（基于行业最佳实践）

### 1. 旅行节奏偏好
- **轻松休闲型** - 不赶行程，慢慢体验，每天1-2个景点
- **充实打卡型** - 希望多看多玩，行程较满，每天3-5个景点
- **深度沉浸型** - 少而精，在一个地方待很久，体验当地生活
- **随性自由型** - 没有固定计划，走到哪玩到哪

### 2. 旅行主题偏好
- **摄影旅拍** - 追求绝美拍摄点，日出日落，星空银河
- **美食探索** - 寻找当地特色美食，米其林，街头小吃
- **历史文化** - 博物馆，古迹，文化遗产，人文故事
- **自然风光** - 山水，森林，湖泊，海岸线
- **城市漫步** - 建筑，街区，咖啡馆，书店
- **户外探险** - 徒步，登山，潜水，露营
- **亲子家庭** - 适合孩子，教育意义，安全舒适
- **浪漫情侣** - 蜜月，纪念日，浪漫场景
- **社交聚会** - 热闹，酒吧，夜市，群体活动

### 3. 旅行风格偏好
- **奢华享受型** - 高端酒店，私人定制，VIP服务
- **品质舒适型** - 中高档住宿，注重体验质量
- **经济实惠型** - 背包客，青旅，性价比优先
- **小众独特型** - 避开人潮，探索冷门，独特体验
- **网红打卡型** - 热门景点，必去清单，社交分享

### 4. 旅行同伴偏好
- **独自旅行** - 一个人，自由独立
- **情侣二人** - 浪漫，私密
- **家庭亲子** - 带孩子，老人
- **朋友结伴** - 三五好友，热闹
- **团队出游** - 公司团建，大团队

### 5. 旅行时间偏好
- **周末短途** - 2-3天，周边游
- **小长假** - 3-5天，黄金周
- **长假深度** - 7天以上，跨省跨国
- **说走就走** - 临时决定，即兴出行

### 6. 交通方式偏好
- **自驾自由** - 租车自驾，灵活自由
- **公共交通** - 高铁，地铁，公交，环保
- **包车省心** - 当地司机，不用自己开
- **徒步骑行** - 慢旅行，低碳环保

### 高频标签分类：
1. **场景类**：#周末去哪儿 #假期去哪玩 #小众旅行地
2. **人群类**：#亲子游 #情侣旅行 #闺蜜游 #一人游
3. **主题类**：#美食之旅 #摄影之旅 #文化之旅 #户外探险
4. **风格类**：#奢华旅行 # backpacking #穷游 #精致游
5. **地域类**：#北京周边 #江浙沪 #云南 #新疆 #川西

### 用户决策因素：
1. **视觉吸引力** - 照片是否好看，是否出片
2. **独特稀缺性** - 是否小众，是否人少
3. **性价比** - 花费是否合理
4. **便利性** - 交通是否方便，是否需要预约
5. **安全性** - 是否适合带孩子/老人
6. **体验丰富度** - 能玩多久，有什么活动
