# 北极雨燕 - 首页 / 探索页设计

## 布局结构

```
┌─────────────────────────────┐
│ 状态栏 (时间/电量)            │
├─────────────────────────────┤
│ 定位 ▾ 北京市              🔍 │  ← 顶部栏
├─────────────────────────────┤
│                             │
│   ┌─────────────────────┐   │
│   │                     │   │
│   │    地图区域          │   │  ← 60% 屏幕
│   │   (可交互地图)        │   │
│   │                     │   │
│   └─────────────────────┘   │
│                             │
├─────────────────────────────┤
│  ○ 推荐路线  ○ 附近景点      │  ← 标签切换
├─────────────────────────────┤
│ ┌─────────────────────────┐ │
│ │ 🏔️ 川西小环线 3日游      │ │  ← 路线卡片
│ │ ⭐ 4.9  📍 成都出发      │ │
│ │ 💰 ¥1,200/人  🚗 自驾    │ │
│ └─────────────────────────┘ │
│ ┌─────────────────────────┐ │
│ │ 🌊 洱海环湖 2日游        │ │
│ │ ⭐ 4.8  📍 大理出发      │ │
│ │ 💰 ¥800/人  🚗 自驾      │ │
│ └─────────────────────────┘ │
├─────────────────────────────┤
│  🗺️    📝    📷    👤      │  ← 底部导航
│  探索   规划   足迹   我的   │
└─────────────────────────────┘
```

## 顶部栏
- **背景**: 白色 + 底部细线 (#E8E8E8)
- **定位按钮**: 左侧，橙色图标 + 城市名
- **搜索**: 右侧放大镜图标，点击展开

## 地图区域
- **默认层级**: 显示当前城市及周边景点
- **景点标记**: 
  - 未选中: 橙色圆点 + 白色边框
  - 选中: 放大脉冲动画，显示景点名称标签
- **路线线条**: 橙色虚线，带箭头方向
- **操作**: 双指缩放、单指拖动、点击标记查看详情

## 标签切换
- **样式**: 胶囊形状，选中填充橙色，未选中灰色背景
- **选项**: 推荐路线 | 附近景点 | 热门摄影点

## 路线卡片
- **尺寸**: 屏幕宽度 - 32px，高度 120px
- **圆角**: 16px
- **阴影**: 轻微投影 (y: 4, blur: 12, opacity: 0.08)
- **图片**: 左侧 40%，圆角裁剪，显示景点代表图
- **信息**: 右侧 60%，标题 + 评分 + 价格 + 标签
- **滑动**: 水平滑动浏览，卡片间间距 12px

## 底部导航
- **背景**: 白色 + 顶部细线
- **图标**: 线性风格，选中填充橙色
- **动效**: 选中时图标轻微上浮 + 缩放

## 色彩应用
- 主按钮/选中: 橙色 (#FF6B35)
- 背景: 浅灰蓝 (#F7F9FC)
- 卡片: 纯白 (#FFFFFF)
- 文字主色: 深灰 (#2D3142)
- 文字次要: 中灰 (#8E92BC)

## 交互细节
- 下拉刷新: 地图区域下拉，显示橙色刷新指示器
- 卡片点击: 轻微缩放 (0.98) 后进入详情
- 地图标记点击: 弹出自定义气泡，显示景点简介
