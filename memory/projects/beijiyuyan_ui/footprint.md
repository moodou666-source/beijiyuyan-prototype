# 北极雨燕 - 足迹页 / 照片地图设计

## 布局结构

```
┌─────────────────────────────┐
│ 我的足迹              ⚙️    │  ← 顶部栏
├─────────────────────────────┤
│                             │

│   │                     │   │
│   │    照片地图          │   │  ← 60% 屏幕

│   └─────────────────────┘   │
│                             │
├─────────────────────────────┤
│  ○ 全部  ○ 公开  ○ 私密      │  ← 筛选标签
├─────────────────────────────┤
│ ┌─────────────────────────┐ │
│ │ 📷 四姑娘山日出          │ │  ← 照片卡片
│ │ 🗓️ 2026.04.15          │ │
│ │ 👁️ 234  ❤️ 56  💬 12   │ │
│ └─────────────────────────┘ │
│ ┌─────────────────────────┐ │
│ │ 📷 洱海星空             │ │

│  探索   规划   足迹   我的   │

```

## 顶部栏
- **背景**: 白色 + 底部细线
- **标题**: "我的足迹"，左侧，加粗
- **设置**: 右侧齿轮图标，点击进入隐私设置

## 照片地图区域
- **地图样式**: 深色模式地图底图，与整体风格统一
- **照片标记**: 
  - 缩略图圆形显示，边框 2px 白色
  - 聚集时显示数字气泡
  - 点击展开为大图预览
- **聚合效果**: 相近照片自动聚合成簇，点击展开
- **时间轴**: 底部小横条，可滑动选择时间段

## 照片标记交互
```
    ┌─────────┐
    │  📷     │  ← 缩略图，圆形
    └────┬────┘
         │
    ┌────┴────┐
    │ 成都·都江堰 │  ← 地点标签
    │ 2026.04.15 │  ← 日期
    └─────────┘
```

## 筛选标签
- **全部**: 显示所有照片
- **公开**: 已分享到照片墙的照片
- **私密**: 仅自己可见
- **样式**: 胶囊切换，选中橙色填充

## 照片卡片
- **尺寸**: 屏幕宽度 - 32px
- **圆角**: 16px
- **图片**: 顶部 70%，16:9 比例
- **信息区**: 底部 30%，白色背景
  - 标题: 拍摄地点，加粗
  - 日期: 灰色小字
  - 统计: 浏览/点赞/评论数
- **隐私标识**: 右上角小锁图标（私密照片）

## 照片墙分享功能
- **开关**: 每张照片右上角切换公开/私密
- **公开照片**: 进入「发现」→「照片墙」板块
- **照片墙布局**: 
  - Pinterest 风格瀑布流
  - 按地点/时间/热度排序
  - 可点赞、评论、收藏

## 交互细节
- 地图区域双指缩放查看照片分布

- 照片支持左右滑动浏览
