## 顶部栏
- **背景**: 白色
- **返回**: 左侧箭头，深灰色
- **标题**: "路线规划"，居中，加粗
- **保存**: 右侧文字按钮，橙色，未填写完整时置灰

## 步骤指示器
- **样式**: 左侧橙色竖线 + 步骤编号圆圈
- **当前步骤**: 圆圈填充橙色，文字加粗
- **已完成**: 圆圈橙色边框 + 内部对勾
- **未完成**: 灰色圆圈

## 时间选择卡片
- **背景**: 白色，圆角 12px
- **阴影**: 轻微投影
- **日期选择**: 点击弹出日历组件
- **日历样式**: 
  - 选中日期: 橙色圆形背景
  - 今天: 橙色边框
  - 范围: 浅橙色背景连接

## 目的地搜索
- **输入框**: 圆角 24px，灰色背景 (#F5F5F5)
- **搜索图标**: 左侧，灰色
- **热门标签**: 下方横向排列，胶囊形状
  - 未选中: 灰色背景 + 深色文字
  - 选中: 橙色背景 + 白色文字

## 偏好选择
- **样式**: 网格布局，2 列
- **单选卡片**: 
  - 未选中: 白色背景，灰色边框
  - 选中: 橙色边框，橙色背景 10% 透明度
  - 图标: 每个偏好有对应 Emoji

## 驾驶时长滑块
- **轨道**: 灰色背景，高度 4px
- **选中段**: 橙色填充
- **滑块**: 圆形，白色 + 橙色边框
- **刻度标签**: 下方显示时长说明

## 生成按钮
- **样式**: 全宽，高度 52px，圆角 26px
- **颜色**: 橙色渐变 (左 #FF6B35，右 #FF8F5A)
- **文字**: 白色，加粗，16px
- **图标**: 左侧火箭 Emoji
- **动效**: 点击时轻微缩放，加载时显示旋转动画


## 结果页面
- **地图视图**: 顶部 50%，显示完整路线
- **路线概览**: 
  - 总里程、预计时间、总花费
  - 每日行程折叠卡片
- **景点列表**: 时间轴形式，显示每个停留点
- **操作按钮**: 
  - "开始导航" (主按钮)
  - "分享路线" (次要按钮)
  - "保存到我的路线"

## 交互细节
- 步骤间自动保存草稿
- 左滑返回上一步
- 地图可切换卫星/标准模式
- 景点可拖拽调整顺序
