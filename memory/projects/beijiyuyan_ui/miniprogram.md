## 适配原则
- 保持与 App 一致的视觉风格
- 针对小程序特性优化交互
- 减少页面层级，提高加载速度

## 顶部栏
- **背景**: 白色
- **Logo**: 左侧，雨燕图标 + "北极雨燕"
- **搜索**: 右侧，点击进入搜索页

## 地图区域
- **简化**: 不显示复杂交互，仅展示当前位置
- **标记**: 附近热门景点标记
- **点击**: 跳转到景点详情

## 快速规划
- **样式**: 三列等分按钮
- **用途**: 根据假期时长快速筛选路线
- **交互**: 点击直接进入规划流程

## 路线卡片
- **简化**: 减少信息密度
- **重点**: 标题 + 一句话描述 + 按钮
- **图片**: 背景图 + 文字叠加

## 小程序特有优化
1. **分享卡片**: 自定义分享图，显示路线概览
2. **订阅消息**: 行程提醒、天气预警
3. **扫码识别**: 扫描景点二维码快速添加
4. **附近的小程序**: 基于地理位置推荐

## 性能优化
- 图片懒加载
- 地图组件按需加载
- 数据本地缓存
- 骨架屏加载状态
