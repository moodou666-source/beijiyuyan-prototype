# 北京雨燕 - 核心定位文档

## 产品定位

**北京雨燕** 是一款以 **摄影打卡出片** 为核心的旅游推荐应用。

## 核心理念

### 1. 摄影优先
- **核心目标**: 帮助用户找到最容易出片、拍照最美的景点和机位
- **用户画像**: 摄影爱好者、自媒体创作者、喜欢拍照打卡的年轻人
- **价值主张**: 不再盲目跟风，每个推荐都经过"出片率"验证

### 2. 真实算法驱动
- **数据来源**: 微博、抖音、小红书真实用户上传的照片
- **核心算法**: 
  - 真实性评分（过滤营销号）
  - 出片率计算（照片质量、点赞互动）
  - 机位推荐（最佳拍摄角度、时间）

### 3. 打卡文化
- **社交属性**: 用户分享打卡照片，形成社区
- **成就系统**: 打卡勋章、摄影等级
- **UGC内容**: 真实用户的照片比官方宣传更可信

## 核心功能

### 1. 智能推荐
```
推荐得分 = 出片率 × 真实性 × 热度 × 距离

其中:
- 出片率: 基于真实用户照片质量计算
- 真实性: 过滤营销号，优先真实用户内容
- 热度: 近期打卡趋势
- 距离: 用户当前位置
```

### 2. 机位导航
- 最佳拍摄点标注
- 拍摄时间建议（黄金时刻、蓝调时刻）
- 样片参考（真实用户作品）
- 器材建议（焦段、滤镜）

### 3. 打卡社区
- 用户上传打卡照片
- 点赞、评论、收藏
- 摄影技巧分享
- 月度摄影大赛

## 目标用户

### 主要用户
1. **摄影爱好者**: 寻找拍摄机位，提升摄影技术
2. **自媒体创作者**: 需要高质量内容素材
3. **旅行打卡族**: 喜欢拍照分享的年轻人
4. **情侣/闺蜜**: 寻找浪漫/有趣的拍照地点

### 次要用户
1. **普通游客**: 想了解哪里拍照好看
2. **本地居民**: 发现身边的美景

## 竞争优势

### vs 传统旅游APP（携程、马蜂窝）
- **他们**: 侧重酒店、机票、攻略
- **我们**: 专注摄影打卡，算法推荐真实出片点

### vs 小红书/抖音
- **他们**: 内容分散，营销号多
- **我们**: 聚合多平台数据，过滤营销号，专注旅游摄影

### vs 500px/图虫
- **他们**: 专业摄影社区
- **我们**: 面向普通用户，降低摄影门槛，提供机位导航
